# Techsector — Premium PDR & Auto Restoration

Demo landing page для Techsector autostudio and PDR service (Almaty).

## Описание

Премиальный automotive landing page в стиле Behance concept для студии кузовного ремонта и PDR (удаление вмятин без покраски).

### Особенности

- 🎨 Тёмный премиальный дизайн
- 🚗 Automotive стиль с красными акцентами
- 📱 Полностью адаптивный layout (desktop + mobile)
- ⚡ Современный tech stack (React + Vite)
- 🎭 Плавные анимации и hover-эффекты
- 📞 Прямые ссылки на WhatsApp и Instagram

### Технологии

- React 18
- Vite
- CSS3 (Grid, Flexbox, Animations)
- Google Fonts (Inter)
- Custom font (Schwarzenegger)

## Установка

```bash
npm install
npm run dev
```

Сайт будет доступен на `http://localhost:5173/`

## Сборка для продакшена

```bash
npm run build
npm run preview
```
